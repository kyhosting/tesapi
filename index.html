// Tentang

Apa itu KY-SHIRO API?

API ini nyambung langsung ke iVAS SMS, support multi-akun biar makin banyak nomor yang bisa dipantau. Cocok banget buat forward OTP ke Telegram bot atau keperluan lain yang butuh kode SMS masuk.

Semua Endpoint

Base URL: https://apishiro.vercel.app

GET /sms Ambil OTP berdasarkan tanggal ▾
Parameter
NamaTipeStatusKeterangan
datestringWAJIBFormat DD/MM/YYYY — tanggal yang dicek
modestringOPSIONALreceived / live / both — default: received
Contoh Request
GET /sms?date=07/03/2026&mode=received
Contoh Response
{ "status": "success", "mode": "received", "total": 5, "accounts_used": 2, "otp_messages": [ { "range": "IVORY COAST 3878", "phone_number": "2250711220970", "otp_message": "Your WhatsApp code: 338-640", "source": "received", "account": "akun1@gmail.com" } ] }
GET /health Cek status login semua akun ▾

Cek apakah API berhasil login ke iVAS. Kalau login: "success" berarti siap terima request.

Contoh Response
{ "status": "ok", "login": "success", "accounts_ok": 2, "accounts_total": 2, "details": [ { "email": "akun1@gmail.com", "login": "success" }, { "email": "akun2@gmail.com", "login": "success" } ] }
GET /accounts List akun terdaftar ▾

Lihat berapa akun yang terdaftar di API. Password tidak ditampilkan.

Contoh Response
{ "total": 2, "accounts": [ { "index": 1, "email": "akun1@gmail.com" }, { "index": 2, "email": "akun2@gmail.com" } ] }
GET /test Cek semua range & nomor ▾
Parameter
NamaTipeStatusKeterangan
datestringOPSIONALFormat DD/MM/YYYY — default: hari ini
Contoh Request
GET /test?date=07/03/2026
GET /test/sms Cek OTP untuk 1 nomor spesifik ▾
Parameter
NamaTipeStatusKeterangan
datestringOPSIONALFormat DD/MM/YYYY
rangestringWAJIBNama range, contoh: IVORY COAST 3878
numberstringWAJIBNomor telepon, contoh: 2250711220970
Contoh Request
GET /test/sms?date=07/03/2026&range=IVORY COAST 3878&number=2250711220970
GET /debug/ranges-raw   /debug/numbers   /debug/sms Debug endpoints ▾

Tiga endpoint khusus buat debug kalau ada yang tidak kedeteksi atau SMS tidak masuk.

Endpoint Debug
EndpointParameter WajibFungsi

/debug/numbersdate, rangeCek nomor dari range tertentu beserta raw response
/debug/smsdate, range, numberCek raw response SMS untuk nomor tertentu
// Cara Tambah Akun

Buka app.py, cari bagian load_accounts(), tambah akun baru di list:

return [ {"email": "akun1@gmail.com", "password": "password1"}, {"email": "akun2@gmail.com", "password": "password2"}, # ← tambah di sini {"email": "akun3@gmail.com", "password": "password3"}, # ← atau di sini ]

Atau pakai environment variable di Vercel: IVAS_ACCOUNTS = email1:pass1,email2:pass2
